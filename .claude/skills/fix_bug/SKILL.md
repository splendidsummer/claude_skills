# Skill: Fix Bug

Fix a bug with minimal, targeted changes.

## Steps
1. Understand the bug - reproduce it if possible.
2. Identify the root cause by examining relevant code.
3. Apply a minimal fix.
4. Add a regression test in `tests/` to prevent future occurrences.
5. Run tests to verify the fix works.
6. Do NOT commit changes unless explicitly requested.

## Output format
- Root cause identified
- Files modified
- Test coverage added
