# Skill: Git Commit

Create a git commit with proper changes.

## Steps
1. Run `git status` to see untracked and modified files.
2. Run `git diff` to review changes.
3. Ask user which files to include in the commit.
4. Draft a commit message following conventional commits format.
5. Stage the files and create the commit.
6. Do NOT push unless explicitly requested.

## Output format
- Files staged
- Commit message
- Commit SHA
