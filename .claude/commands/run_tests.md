# Run Tests

Run the test suite and report results.

## Steps
1. Check for test files in `tests/` directory.
2. Run pytest with appropriate options.
3. Report test results including passed, failed, and skipped tests.
4. If failures exist, report which tests failed and why.

## Output format
- Test summary (passed/failed/skipped)
- List of any failures with error messages
- Coverage report if available
